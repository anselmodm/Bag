Bag

We have a bag that allows us to store objects of the same type. This bag has a small zipper,
so it's not possible to see what's inside. Therefore, when an item is removed, a random item is removed, with equal probability among them.

In this activity, a Bag class should be created that allows us to store the unordered collection of objects.
If the bag is full, the extra items should be ignored.

To obtain a random number, we can use the Random class from Java, for example.
